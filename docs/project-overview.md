# Project Overview

## Directory Structure

- `./` the root directory contains configuration files for the project for e.g. TypeScript, ESLint, Webpack etc.
- `assets/` holds files that are used by all components in the project as is. E.g. images, fonts, translations, audio files etc.
- `docs/` holds the project's documentation.
- `src/` holds the project's source code in TypeScript, JSON and SASS.
- `test/` contains unit and snapshot tests for the code in `src/`.
